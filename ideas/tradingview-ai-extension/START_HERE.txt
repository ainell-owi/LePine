WHAT YOU NEED TO DO:
────────────────────

STEP 1: Create 3 icon files (5 minutes)
   → See QUICK_START.md for easiest methods
   → Put them in the "icons" folder

STEP 2: Load extension in Chrome/Brave (2 minutes)
   → Go to chrome://extensions/
   → Turn on "Developer mode"
   → Click "Load unpacked"
   → Select this folder

STEP 3: Get Anthropic API key (5 minutes)
   → Go to https://console.anthropic.com/
   → Sign up, create API key
   → Add $5-10 in credits

STEP 4: Configure extension (1 minute)
   → Click extension icon
   → Paste API key
   → Click "Save"

STEP 5: Test on TradingView (2 minutes)
   → Go to TradingView.com
   → Open any chart
   → Look for "🤖 Analyze Chart" button
