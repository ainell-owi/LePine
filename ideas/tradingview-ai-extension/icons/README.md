# Extension Icons

This directory should contain the following icon files:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Creating Icons

You can create icons using:
- Image editing software (Photoshop, GIMP, etc.)
- Online icon generators
- AI image generators

The icons should represent the extension (AI/robot theme, TradingView, Pine Script, etc.)

For now, you can use placeholder images or create simple icons with:
- Text: "TV AI"
- Symbol: 🤖 (robot emoji converted to image)
- Colors: Purple gradient (#667eea to #764ba2) matching the extension theme
